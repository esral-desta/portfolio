GTA keyboard_bot

This Telegram bot is a great tool for users who want to communicate in Ge'ez,Tigrgna and Amharic , but are more comfortable typing in Latin alphabets. By using this bot, you can easily type in the Latin script and have it automatically converted to Ge'ez, Tigrgna, and Amharic. This will save your time and make it easier to communicate with people who are more fluent in these languages. this bot helps to bridge the gap between different writing systems, allowing users to communicate more effectively and efficiently. It is a convenient and user-friendly solution that eliminates the need for users to learn different scripts or install additional software. this bot is a great example of how technology can be used to make communication more inclusive and accessible for everyone.
